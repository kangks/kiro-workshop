# Product Overview

Micro Blogging App — a Twitter/X-style social media platform where users can create short-form posts (280 char limit), follow other users, like posts, and browse a feed.

## Core Features
- User registration and authentication (Cognito-backed)
- Create, read, and like posts
- User profiles with bio and avatar
- Follow/unfollow users
- Feed with sorting (newest, popular) and pagination
- User profile pages showing their posts and follower counts

## Design Language
See `kiro-workshop/DESIGN_LANGUAGE.md` for the full design system. Key points:
- Purple (#8b5cf6) accent color on white/light-gray backgrounds
- Fully rounded buttons (border-radius: 9999px)
- System font stack
- Mobile-first responsive design
- Social media card-based layout
